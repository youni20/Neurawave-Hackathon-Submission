# Open Seizure Detector API

This is an open-source alternative that you can continue working on after the hackathon.

The OpenSeizureDetector project provides a web API for managing health-related events and data, similar to migraine tracking use cases.

**Website:** https://www.openseizuredetector.org.uk/

**GitHub Repository:** https://github.com/OpenSeizureDetector/webApi/tree/master/api

Since this is an open-source project, you can fork it, modify it, and build upon it for your own purposes.
